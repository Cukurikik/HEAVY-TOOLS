# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/aa40f60b-1ab6-45b9-b601-2e1e7294e586

## Setup

1. Clone this repository: `git clone https://github.com/ikyy/HEAVY-TOOLS.git`
2. Set the `sk-e391a1e39ed048e1ac26d158b379b857` in [.env.local](.env.local) to your Alibaba Cloud API key for Qwen models
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
