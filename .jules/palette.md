## 2026-03-21 - Interactive Elements Focus and ARIA
**Learning:** Found instances where interactive semantic elements (like `<button>`) were replaced by generic `<div>` elements or lacking essential `aria-label` attributes and keyboard focus styling.
**Action:** Ensure all interactive elements utilize semantic HTML tags, include `aria-label` when icons are used without accompanying text, and implement explicit `focus-visible` states for keyboard navigation accessibility.
